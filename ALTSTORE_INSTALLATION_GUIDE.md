### Step 1: Get AltStore (First Time Only - 5 minutes)

**On their iPhone:**

1. **Open Safari browser** (the blue compass icon)

3. **Scroll down** and find the big **"Download AltStore"** button

### Step 3: Install ExpenseSettle App (3 minutes)

**On their iPhone:**

1. **Open AltStore app** (the blue icon on home screen)

3. **Click the "+" button** (top left)
4. **AltStore opens file picker**
5. **Navigate to Downloads folder**
6. **Select "ExpenseSettle.ipa"** file
7. **AltStore starts installing:**
   - "Fetching metadata..."
   - "Building..."
   - "Installing..."
   - Wait 2-3 minutes
8. **Installation complete!** âœ…

### App Refreshes Every 7 Days

This is a limitation of the free tier:
- **After 7 days**, the app will show a message
- **No need to reinstall!**
- **Just open AltStore**
- **Tap the app** â†’ **"Refresh"** button
- **Takes 30 seconds**

3. Find "ExpenseSettle"
4. Tap "Refresh"

## ğŸ› Troubleshooting

### "AltStore won't install from website"

1. Make sure they're on Safari (not Chrome)
2. Go to altstore.io again
3. Look for "Download AltStore" button
4. Make sure they enabled AltStore in Settings first

### "Cannot select IPA in AltStore"

1. Make sure IPA was fully downloaded (check download percentage)
2. Try downloading again
3. Make sure file is in Downloads folder

### "Installation fails during install"

1. Make sure WiFi is connected
2. Make sure Apple ID password is correct
3. Try again in AltStore

### "App shows as expired after 7 days"

This is normal! Just refresh:
1. Open AltStore
2. Go to Apps tab
3. Tap the ExpenseSettle app
4. Click "Refresh"
5. Done!
