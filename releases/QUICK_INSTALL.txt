METHOD 1: Using ADB (Computer + USB Cable - FASTEST)
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
1. Connect phone to Mac/Windows computer via USB
2. Enable USB Debugging on phone:
   - Settings > Developer Options > USB Debugging (turn ON)
   - Tap "Allow" for USB debugging permission
3. On computer, open terminal and run:

   adb devices              (to verify phone is connected)
   adb install ExpenseSettle-v1.0.0-debug.apk

4. Done! Open the app on your phone
